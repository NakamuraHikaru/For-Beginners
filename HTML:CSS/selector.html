ここでは、３つのクラスを定義しています。
新しいクラス名を作ることなく、HTML/CSSを用いて以下の問題を考えてみましょう。

content{ background-color: orange; }
content-gy{ background-color:greenyellow; }
content-cyan{ background-color:cyan; }
＊'.new-content{background-color:~;}'とか'<div class="new-content">new content</div>'とか宣言しちゃだめです。

Level１

同じクラスでも親子の関係で色が変わる

content
content
content-cyan
content 直接の子（直下）ではなくても色が変わる

クラスを２つ使って３色つくる

Level3

同じ階層である要素の直後に隣接していると色が変わる

Level2の要素

Level4

Level5

content
